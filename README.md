# iPadBox
# 奥行き錯覚テスト（Face-Tracked Viewer）

このリポジトリは、**Webブラウザ上で動作する奥行き錯覚デモ**のための機能テスト用コードです。  
iPad（Safari）を主な対象とし、カメラで取得した顔の位置に応じて、  
箱の中を「覗き込んでいるように見える」視点移動を行います。

コミケC107で配布した冊子→https://wasuke.github.io/iPadBox/c107FreePaper.pdf

---

## 重要な位置づけ

- 本プロジェクトは **研究・教育用途の機能検証用デモ**です  
- **完成品・製品レベルのアプリケーションではありません**
- **学生向け教材・実験素材**としての利用を想定しています

アルゴリズムや実装は「分かりやすさ」と「改造しやすさ」を優先しています。

---

## 主な機能

- Webカメラ（内蔵／外付けUSB）による顔トラッキング  
- 最初に検出した顔をロックし、背後を人が通過しても追従を維持  
- 顔の画面内位置に応じたカメラ移動（覗き込み錯覚）  
- 顔が近づくとズーム（起動時の顔サイズを基準）  
- 市松模様＋明るいパッチを持つ箱状空間  
- GLB（glTF）モデルの表示  
  - 中央に小さめで浮遊
  - ゆっくり自動回転
- 強力なスポットライト（白飛びあり）

---

## 想定環境

- iPad（USB-C / Lightning）＋ Safari  
- macOS / Windows（Chrome, Safari）  
- HTTPS 環境（GitHub Pages / Live Server など）

※ カメラ使用のため **HTTPS または localhost** が必要。

---

## 使い方（概要）

1. このリポジトリを GitHub Pages またはローカルサーバで公開  
2. `index.html` をブラウザで開く  
3. カメラアクセスを許可  
4. 画面中央に顔を置いて体験
5. DemoPage→https://wasuke.github.io/iPadBox/

GLBモデルを表示する場合は、URL パラメータで指定します。
